Lara-Drive
=========

## Demo video: https://youtu.be/V90VHmWr0Rc

## Trello: https://trello.com/b/HMXKgWsH/laradrive


* Application using Laravel PHP and MySQL local database with authentication for FILE:
    - uploading 
    - text file creating 
    - listing
    - filtering 
    - searching 
    - ordering 
    - editing 
    - downloading 
    - deleting 
    - sending with mailing (mailtrap)
